## Slynk Lite | File Upload Sharing API

This project is open source under Apache 2.0.

If you fork or adapt it, please keep a small reference to the original project
and author where it makes sense, such as in your README, about page, or NOTICE
file. The intent is simple attribution, not friction.

You are welcome to describe your work as "based on Slynk Lite | File Upload Sharing API" or
"forked from Slynk Lite | File Upload Sharing API by Liem TRAN (liemtv96)".

Please avoid presenting your fork as the original official project unless it is
actually maintained by the original author.
